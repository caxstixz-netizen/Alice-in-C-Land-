# Alice-in-C-Land-

Alice in C++ Land is a text-based educational adventure game built using C++ and Object-Oriented Programming (OOP). The game combines storytelling with programming challenges where the player explores Wonderland while learning core C++ concepts.

Players take the role of Alice and progress through multiple stages such as the Hall of Knowledge, Animal Encounters, the Maze, the Mad Hatter's Debugging Challenge, and the Queen's Final Battle. Each stage introduces programming ideas like classes, inheritance, polymorphism, abstraction, and encapsulation through quizzes, puzzles, and character interactions.

## Project Objective

The objective of this project is to demonstrate how gamification can make learning programming more engaging. Instead of only reading theory, players actively solve problems and interact with the game environment to reinforce their understanding of C++ and OOP.

## Game Stages

Hall of Knowledge  
Alice learns basic C++ concepts and answers quiz questions to progress.

Animal Encounter  
Talking animals introduce deeper programming concepts and insights.

Maze Challenge  
Alice must answer questions correctly to escape a maze.

Mad Hatter Debugging Challenge  
Players debug incorrect C++ code snippets.

Queen's Final Battle  
The final challenge where Alice answers advanced questions to defeat the Queen.

## OOP Concepts Implemented

Encapsulation – Data and functions are organized within classes.  
Inheritance – The Alice class integrates functionality from other classes representing different stages.  
Polymorphism – Different characters provide different responses through multiple functions.  
Abstraction – Game mechanics hide complex logic behind simple class interactions.  
Nested Classes – Question structures are used inside specific modules like Maze and Queen.
